This project simulates a steam turbine cycle using **CoolProp** for thermodynamics and **SciPy** for numerical solving. It calculates the required mass flow rate to achieve a target power output, accounting for non-linear efficiency curves.

## Prerequisites

*  Python 3.8 or higher

## Installation

1. **Clone or download** this repository.
2. Open your terminal/command prompt in the project folder.
3. Install the required libraries:

```bash
pip install -r requirements.txt
```

## Run a script

1. Open your terminal/command prompt in the project folder.
2. Type:
```bash
py folder\script.py
```
